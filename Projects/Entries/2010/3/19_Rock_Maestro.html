Rock Maestro

 

Done as the final project for 6.813, MIT's user interface class, this project was meant to explore an interesting kind of interaction. Our concept originated from Rock Band and Guitar Hero, and our project aimed to be a creation tool for the songs in Rock Band. This user interface was designed to allow people to create the songs to hypothetically be used in Rock Band or Guitar Hero. Done in Adobe Flex

 

Spring 2010

6.813 User Interfaces Final Project

Jennifer Chan, Kimberly Baldauf, Nicole Holm
